GOOGLE FONTS Google Fonts

Google Fonts is a computer font and web font service owned by Google. This includes free and open source font families, an interactive web directory for browsing the library, and APIs for using the fonts via CSS and Android. Google Fonts was launched in 2010[5] and revamped in 2011,[6] 2016,[7] and 2020.[8] On March 3, 2020, Google updated the catalog website with support for variable fonts.[citation needed] On March 2, 2021, the Google Fonts team announced they were adding support for open source icons.[9]
